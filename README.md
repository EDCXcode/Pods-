# CocoaPods使用
## 一.安装CocoaPods<br>
1.官方教程<br>
https://guides.cocoapods.org/

2.推荐最新安装教程<br>
http://www.jianshu.com/p/b64b4fd08d3c

## 二.已经安装好了CocoaPods
1.启动终端进入你的项目路径  cd 项目路径 回车 <br>

2.pod init <br>

3.现在你的项目就会有Podfile文件，点击编辑你要用的三方库  <br>

例如 pod 'AFNetworking', '~>3.0.4' <br>

4.pod install <br>

5.表示集成成功 Pod installation complete!
